# Задание 1: TCP Чат

## Описание
Простой многопоточный TCP Чат Сервер и Клиент с использованием Windows Sockets (Winsock2).

- **Сервер**: Слушает порт 8888. Принимает множество клиентов. Рассылает полученные сообщения всем остальным подключенным клиентам.
- **Клиент**: Подключается к localhost:8888. Отправляет ввод пользователя на сервер и выводит сообщения, полученные от сервера.

## Сборка и Запуск

### Требования
- MinGW-w64
- Wine

### Использование скрипта
```bash
./run.sh
```
Это соберет проект, запустит сервер в фоновом режиме и запустит одного клиента в текущем терминале.
Чтобы протестировать с несколькими клиентами, откройте новый терминал, перейдите в эту директорию и запустите:
```bash
wine chat_client.exe
```

### Ручная сборка
```bash
make
# Терминал 1
wine chat_server.exe
# Терминал 2
wine chat_client.exe
```
