# Задание 1: Самовосстанавливающийся процесс

## Описание
Приложение демонстрирует процесс, который перезапускается с сохранением состояния (счетчик и текстовое сообщение). При закрытии процесс сохраняет данные в разделяемую память, создает дочерний процесс и завершается. Новый процесс загружает сохраненное состояние и продолжает работу.

## Сборка
```bash
# CMake (рекомендуется)
cmake -B build -DCMAKE_TOOLCHAIN_FILE=../toolchain-mingw64.cmake
cmake --build build

# Или Makefile
make
```

## Запуск
```bash
wine build/self_restoring.exe
# или
make run
```

## Использование
1. Введите текст в поле "Message"
2. Нажмите "Increment Counter" для увеличения счетчика
3. Нажмите "Close (Self-Restore)" для перезапуска процесса
4. Новое окно откроется с сохраненными значениями

## Технические детали
- **Разделяемая память:** `CreateFileMapping()`, `OpenFileMapping()`, `MapViewOfFile()`
- **Создание процесса:** `CreateProcess()` с путем из `GetModuleFileName()`
- **Структура данных:** `AppContext` (counter, message, isRestarting)
